# HAKARI - 買い物判定アプリ

感情に頼らず、客観的に「買うべきか」を判定するコンソールアプリ


**手動で実行**
javac *.java
java -Dfile.encoding=UTF-8 HakariConsole

## 判定の流れ

1. 慎重ライン（これ以上の金額は慎重に考える基準）を設定
2. 価格情報を入力（実売価格・相場・セール有無）
3. 商品ジャンルを選択（食品 / 美容品 / ガジェット / ゲーム / 娯楽品 / プレゼント）
4. 購入動機を選択
5. 詳細条件を入力（使用頻度・期待効果・代替不可度など）
6. GO / STOP の判定結果を表示

## スコア計算の仕組み

最終スコア = 基本スコア + セールボーナス + リターン合計 - リスク合計

基本スコア：相場より安いほどプラス
セールボーナス：割引率に応じて加算
リターン合計：使用頻度・期待効果・代替不可度・永続性から算出
リスク合計：失敗リスク・管理コスト・慎重ライン超過から算出

スコアが **0以上 → GO（買い）**、**0未満 → STOP（見送り）**
